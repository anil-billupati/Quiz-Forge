# Sample user import files

These files can be used to test the Org Admin **Bulk Import Participants** feature at:

**UI:** `/org-admin/users/import`  
**API:** `POST /users/bulk`

## Files

- `sample-users.csv` — CSV with header `email,first_name,last_name`
- `sample-users.json` — JSON array of the same users

All addresses use the reserved `.com` TLD so they will not be delivered to real inboxes.
